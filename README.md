# Nocturnal Flight Call Resources

Scripts, tools, and notes for nocturnal flight call recording, analysis, and research.

This repository is intended for birders, recordists, researchers, and tech-curious beginners who want to record and analyze nocturnal flight calls.

## Start here
* Getting started (text doc)
* Equipment (folder called “hardware” → text doc)
* Software, code, and analysis (folder called “software” → text doc, code)
* Recording protocols (text doc)
* Publications and guides
* Glossary (text doc)

Supporting PDFs, images, and sample files are stored in the `reference/` folder when redistribution is appropriate. Some resources may be linked rather than uploaded because of copyright or licensing limits.
