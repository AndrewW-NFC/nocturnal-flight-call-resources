# Nocturnal Flight Call Resources

Scripts, tools, and notes for nocturnal flight call recording, analysis, and research.

This repository is intended for birders, recordists, researchers, and tech-curious beginners who want to record, organize, or analyze nocturnal flight calls.

## Tools

Code and scripts are stored in the `tools/` folder.

- `tools/nfc-recording-quality-forecast/`: Weather-based planning tools for estimating likely NFC recording quality.
- `tools/audition-hourly-recording/`: Scripts for automating hourly recording workflows in Adobe Audition.

## Reference files

Supporting PDFs, images, and sample files are stored in the `reference/` folder when redistribution is appropriate. Some resources may be linked rather than uploaded because of copyright or licensing limits.
