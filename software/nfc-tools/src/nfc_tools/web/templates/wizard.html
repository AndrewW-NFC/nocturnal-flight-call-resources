Welcome

Let's get you set up to record nocturnal flight calls overnight. This takes about 3 minutes.

1. Where are you?

Your location helps the analyzer pick which species are likely, and is used for weather logs.

Timezone: {{ cfg.site.timezone }}

2. Pick your microphone

Select the input you want to record from. Then click "Test" to make sure it's hearing something.

3. When should it record?

Defaults are good for most folks (overnight, evening to morning). You can also pick a sunset/sunrise-based preset.

4. Analyzers

These run automatically on each recording. They need to download some files (a few hundred MB) the first time.
